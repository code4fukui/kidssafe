# キッズセーフ / KidsSafe

地域の安全マップを作成し、スマートフォンやパソコンで共有できるツールです。Excel/Numbersでデータを編集しアップロードすることで、地図を簡単に更新できます。

## デモ

- [越前市国高地区](https://code4fukui.github.io/kunitaka)
- [福井市棗地区](https://code4fukui.github.io/kidssafe-natsume)
- [越前市岡本地区](https://code4fukui.github.io/kidssafe-okamoto)

## 機能

- 地図上に安全情報を表示
- CSVデータの編集・アップロードによる地図の更新
- QRコード表示による地図共有

## 使い方

1. [kidssafe-template](https://github.com/code4fukui/kidssafe-template/)をベースに新しいリポジトリを作成
2. 地区に合わせてREADME.md、アプリ設定、データを変更
3. GitHub Pagesで公開

データの更新方法など、kidssafe-templateの説明を参照してください。
